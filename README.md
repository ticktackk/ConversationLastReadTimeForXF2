Conversation Last Read Time for XenForo 2.1.7+
==============================================

Description
-----------

This add-on adds "Last Read Date" next to each conversation participants.

Requirements
------------

- PHP 7.3.0+

Permissions
-----------

#### Conversation permissions

- View last read time
